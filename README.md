**PreSkool** est une plateforme ERP/LMS scolaires de nouvelle génération. Ce projet transforme l'expérience académique traditionnelle en un environnement numérique **Premium**, **Intelligent** et **Collaboratif**. 

---

## 💎 FONCTIONNALITÉS D'EXCELLENCE

### 🤖 Intelligence Artificielle & UX
- **Système de Chatbot IA** : Assistant contextuel pour les étudiants, professeurs et administrateurs.
- **Premium Dark Mode** : Confort visuel total avec un design élégant et moderne.
- **OmniSearch** : Moteur de recherche global pour une navigation instantanée.

### 🆔 Gestion & Identité Numérique
- **Attendance via QR Code Dynamique** : Système de pointage en temps réel avec renouvellement de sécurité toutes les secondes.
- **Cartes Étudiantes Digitales** : Génération automatique de cartes avec photo et groupe sanguin.
- **Générateur d'Attestations** : Création professionnelle de documents officiels (Certificats de scolarité, etc.).

## 🚀 INSTALLATION RAPIDE

1.  **Clonage & Environnement** :
    ```bash
    git clone https://github.com/Homam-Dany/Application_Web_Systeme_De_Gestion_Scolaire.git
    cd Application_Web_Systeme_De_Gestion_Scolaire
    python -m venv venv
    ./venv/Scripts/activate # Windows
    ```
2.  **Configuration** :
    ```bash
    pip install -r requirements.txt
    python manage.py migrate
    ```
3.  **Lancement** :
    ```bash
    python manage.py runserver
    ```
Accès : `http://127.0.0.1:8000/`
